# Sound Files

This directory contains sound effects for the app.

## Required Files

1. **ambient-calm.mp3** - A calm, ambient background sound for the home screen
   - Suggested: Soft nature sounds, gentle piano, or ambient music
   - Should be loopable (seamless loop)
   - Recommended duration: 30-60 seconds
   - Volume will be set to 30% by default

2. **wooden-click.mp3** - A subtle wooden click sound for interactions
   - Suggested: Soft wood tap or click sound
   - Should be short (< 0.5 seconds)
   - Volume will be set to 50% by default

## Where to Find Sounds

Free sound resources:
- Freesound.org - https://freesound.org
- Pixabay Sounds - https://pixabay.com/sound-effects
- YouTube Audio Library
- Epidemic Sound (requires subscription)

## File Placement

Place the audio files directly in this directory:
- `/public/sounds/ambient-calm.mp3`
- `/public/sounds/wooden-click.mp3`

The app is configured to load these files automatically.
